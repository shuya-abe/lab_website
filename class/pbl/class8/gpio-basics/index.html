GPIO基礎と回路の考え方

概要

GPIOはRaspberry Piでセンサ入力やアクチュエータ出力を扱うための基本I/Oです。電子回路の最小知識（電圧・電流・抵抗）と合わせて理解します。

手順

1. GPIOの役割を理解する

  1. GPIOは入力にも出力にも使える。
  2. どのピンを使うかは事前に決める。

2. 回路の基本を押さえる

  1. 電圧（V）、電流（I）、抵抗（R）を確認する。
  2. LEDには抵抗を直列で入れる。
  3. GNDへの戻り経路を必ず作る。

3. 配線前後の安全確認をする

  1. 配線時は電源OFFで作業する。
  2. 配線後に極性とピン番号を再確認する。

コード

# ピン配置の確認
pinout
# コメント: Raspberry Pi上でGPIOの対応表を確認できる

# PythonでGPIOを扱う準備（実行例）
python3 - <<'PY'
import RPi.GPIO as GPIO

GPIO.setmode(GPIO.BCM)      # コメント: 物理ピン番号ではなくBCM番号を使う
GPIO.setup(2, GPIO.OUT)     # コメント: GPIO2を出力として設定
print("GPIO2 ready")
GPIO.cleanup()              # コメント: 終了時に設定を解放する
PY

解説

  • GPIO番号の指定方式（BCM/BOARD）を混在させないことが重要です。
  • 回路は「行き（電源側）と戻り（GND側）」の両方が必要です。
  • GPIO.cleanup() を入れると、次の実行で不安定になりにくくなります。

よくあるミス

  • BCM番号と物理ピン番号を混同する。
  • LEDの極性（長い足がプラス側）を逆に挿す。
  • 抵抗なしで接続して部品を痛める。
